# E-Commerce Website - FPT Shop

## Mô Tả Dự Án

Dự án xây dựng website thương mại điện tử FPT Shop - một nền tảng mua sắm trực tuyến chuyên về các sản phẩm công nghệ, điện thoại, laptop, máy tính bảng và phụ kiện điện tử. Website cung cấp trải nghiệm mua sắm hiện đại, tiện lợi với giao diện thân thiện, dễ sử dụng.

### Tính Năng Chính

- **Quản lý sản phẩm**: Hiển thị danh mục sản phẩm đa dạng với thông tin chi tiết, hình ảnh và giá cả
- **Giỏ hàng**: Thêm, xóa, cập nhật số lượng sản phẩm trong giỏ hàng
- **Tìm kiếm & Lọc**: Tìm kiếm sản phẩm theo tên, phân loại theo danh mục, giá, thương hiệu
- **Quản lý người dùng**: Đăng ký, đăng nhập, quản lý thông tin cá nhân
- **Thanh toán**: Xử lý đơn hàng và các phương thức thanh toán
- **Quản trị viên**: Quản lý sản phẩm, đơn hàng, khách hàng

### Công Nghệ Sử Dụng

- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Backend**: PHP
- **Database**: MySQL
- **Version Control**: Git, GitHub
